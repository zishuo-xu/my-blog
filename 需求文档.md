### 三、后端接口要求

| `/api/v1/public/articles/{id_or_slug}` | GET | 根据ID/permalink获取文章详情，访问时自动累加阅读量 |

| `/api/v1/public/tags` | GET | 获取所有标签+对应文章数量 |
| `/api/v1/public/archive` | GET | 获取按年/月聚合的归档数据 |
| `/api/v1/public/search` | GET | 全文搜索文章 |
| `/api/v1/public/sitemap.xml` | GET | 生成sitemap |

| 接口 | 方法 | 功能说明 |
|------|------|----------|
| `/api/v1/admin/login` | POST | 账号密码登录，返回JWT |
| `/api/v1/admin/articles` | GET | 获取所有文章（含草稿），支持筛选、搜索 |

| `/api/v1/admin/articles/{id}` | PUT | 更新文章内容、发布状态 |
| `/api/v1/admin/articles/{id}` | DELETE | 删除文章，支持可选删除关联图片 |
| `/api/v1/admin/categories` | POST/PUT/DELETE | 分类增删改 |
| `/api/v1/admin/tags` | POST/PUT/DELETE | 标签增删改 |
| `/api/v1/admin/upload/image` | POST | 图片上传，支持多图 |
| `/api/v1/admin/export` | GET | 导出所有文章+图片为ZIP包 |
| `/api/v1/admin/import` | POST | 批量导入ZIP包的MD文章 |
| `/api/v1/admin/backup` | GET | 一键备份数据库+所有图片 |
